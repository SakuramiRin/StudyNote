包名：
	多单词组成时所有字母都小写
类名
	多单词首字母大写
变量名和函数名
	第一个字母小写后面大写
常量名：
	所有字母都大写且单词间用下划线_



java  语言基础组成
	关键字	标识符	注释 	常量和变量	运算符	语句	函数	数组

		关键字
			定义数据类型：
				class interface byte short int long float double char boolean void
			定义数据类型值的关键字：
				true false 	null
			定义流程控制
				if else swith case defalut while do for break continue return
			定义权限修饰符：
				public private protect
			定义类、函数、变量修饰符：
				abstract final static synchronized
			定义类与类之间关系：
				extends implements
			定义建立实例及引用实例、判断实例的关键字：
				new this super instaceof(是否属于该类--->多态)
			异常处理：
				try catch finally throw throws
			包：
				package import
			其他修饰符关键字：
				native【本地方法，看不到方法体】 strictfp transient【不能被序列化】 volatile assert


		标识符：
			英文字母 数字(不可开头) _ $
