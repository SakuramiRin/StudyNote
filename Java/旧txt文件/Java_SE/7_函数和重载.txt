定义(方法)：
  明确好运算结果是什么
  明确是否需要未知的内容参与运算

格式：
  修饰符 返回值类型 函数名(参数类型 形式参数1，参数类型 形式参数2){
    return 返回值；
  }

特点：
  定义函数可以将功能代码进行封装
  便于复用
  只有被调用才会被执行
  提高了代码的复用性
    注意：
      函数只能调用函数，不可以在函数内部定义函数。
-----------------------------------------------
重载：
  在同一个类中，允许存在一个以上的同名函数，只要他们的参数个数或者参数类型不同，且和返回值类型无关！！

  何时使用：功能相同，但参与运算的未知内容不同。

  特点：与返回值类型无关，只看参数列表

  好处：方便阅读，优化程序设计
